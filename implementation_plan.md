# 🧠 맞춤형 기술/업무 트렌드 큐레이터 (TrendPulse)

사용자가 설정한 직무(기구 설계, 자동화, IT 등)에 맞춰 최신 기술 동향, 해외 논문, 특허 소식, 교육 정보를 AI가 매일 아침 요약 브리핑해 주는 싱글페이지 웹 앱입니다.

## User Review Required

> [!IMPORTANT]
> **데이터 소스 방식**: 실제 API 연동(Google Scholar, USPTO 등)은 API 키 및 백엔드 서버가 필요합니다. 본 계획에서는 **프론트엔드 전용(HTML/CSS/JS)** 으로 구현하며, **시뮬레이션 데이터 + 데모 UI**로 완성도 높은 프로토타입을 제작합니다. 추후 실제 API 연동은 백엔드 추가 시 확장 가능합니다.

> [!IMPORTANT]
> **AI 요약 기능**: 실제 LLM API 호출 대신, 각 직무별로 사전 준비된 고품질 트렌드 데이터를 기반으로 브리핑을 생성합니다. 향후 OpenAI/Gemini API 연동으로 실시간 요약이 가능하도록 구조를 설계합니다.

- **반응형**: 모바일/태블릿/데스크탑 브레이크포인트 지원

#### 2. 온보딩 시스템 (JS)

- 첫 방문 감지 (`localStorage` 기반)
- 3단계 온보딩 모달:
  1. 환영 화면 + 앱 소개
  2. 직무 카테고리 선택 (6개 중 택 1~3)
  3. 세부 관심 키워드 태그 입력
- 설정값 `localStorage`에 저장
- 온보딩 완료 후 메인 대시보드로 부드러운 전환

#### 3. 모닝 브리핑 대시보드 (JS)

- **실시간 시계** + 날짜 (한국어 포맷)
- **AI 인사말**: 시간대별 맞춤 인사 ("좋은 아침이에요! 오늘의 기술 트렌드를 준비했습니다 🚀")
- **핵심 지표 카드** (4개):
  - 📈 오늘의 트렌드 수
  - 📄 새로운 논문 수
  - 💡 최신 특허 수
  - 🎓 추천 교육 수
- 숫자 카운트업 애니메이션

#### 4. 컨텐츠 데이터 시스템 (JS)

- 6개 직무별 사전 준비된 트렌드 데이터:
  - 기술 동향 (각 8~10개 항목)
  - 논문 (각 5~8개 항목)
  - 특허 (각 5~8개 항목)
  - 교육 정보 (각 4~6개 항목)
- 각 항목에 포함되는 필드:
  - 제목, 요약, 소스, 날짜, 중요도 점수(1-100), 카테고리 태그, 관련 키워드
- 사용자 설정 직무에 따라 동적 필터링

#### 5. 트렌드 피드 UI (HTML + CSS + JS)

- **카드 기반 레이아웃**: 글래스모피즘 카드
- **중요도 배지**: 🔴 Critical(90+) / 🟠 High(70-89) / 🟡 Medium(50-69) / 🟢 Low(<50)
- **카테고리 태그 필터**: 클릭으로 필터링
- **카드 확장**: 클릭 시 상세 내용 + AI 요약 표시
- **무한 스크롤 느낌**: 스크롤 시 추가 카드 로딩 애니메이션

#### 6. 논문 브리핑 섹션 (HTML + CSS + JS)

- 논문 카드: 제목, 저자, 저널/학회, 발행일, 핵심 요약
- **인용 지수** 시각화
- **관련도 점수** 프로그레스 바
- 원문 링크 (데모용 #)

#### 7. 특허 소식 섹션 (HTML + CSS + JS)

- 특허 카드: 특허번호, 출원인, 기술분야, 핵심 청구항 요약
- **타임라인 뷰**: 최근 특허 시간순 정렬
- 기술 분류 태그

#### 8. 교육 추천 섹션 (HTML + CSS + JS)

- 교육 카드: 강의명, 플랫폼, 난이도, 예상 소요시간, 관련 기술
- **난이도 표시**: ⭐ 초급 / ⭐⭐ 중급 / ⭐⭐⭐ 고급
- 수강 링크 (데모용 #)

#### 9. 트렌드 시각화 (JS - Canvas)

- **레이더 차트**: 선택 직무의 기술 영역별 트렌드 점수
- **바 차트**: 주간 트렌드 변화
- 순수 Canvas API로 구현 (외부 라이브러리 없음)

#### 10. 북마크 시스템 (JS)

- 각 카드에 북마크 아이콘
- `localStorage` 기반 저장
- 북마크 목록 사이드 패널

#### 11. 설정 패널 (HTML + CSS + JS)

- 직무 카테고리 변경
- 관심 키워드 수정 (태그 입력)
- 다크/라이트 모드 토글
- 브리핑 시간 설정 (데모)
- 데이터 초기화

#### 12. 사이드바 네비게이션 (HTML + CSS + JS)

- 아이콘 + 텍스트 네비게이션
- 활성 섹션 하이라이트
- 접기/펼치기 토글
- 모바일: 햄버거 메뉴 → 슬라이드 오버레이

## 기술 스택 상세

| 항목 | 선택 | 이유 |
|------|------|------|
| **구조** | HTML5 | 시맨틱 마크업 |
| **스타일** | Vanilla CSS | 글래스모피즘 + CSS 변수 기반 테마 |
| **로직** | Vanilla JS (ES6+) | 외부 의존성 없이 순수 구현 |
| **폰트** | Pretendard + Inter | 한/영 가독성 최적 조합 |
| **아이콘** | SVG 인라인 | 외부 의존성 없음, 크기/색상 자유 제어 |
| **차트** | Canvas API | 라이브러리 없이 커스텀 차트 |
| **저장소** | localStorage | 설정/북마크 영속 저장 |
| **배포** | 단일 HTML | 파일 더블클릭으로 즉시 실행 |
